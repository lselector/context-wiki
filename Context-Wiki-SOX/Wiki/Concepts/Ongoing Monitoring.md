Ongoing monitoring comprises monitoring procedures built into an organization's routine operations — supervisory activities, trend analysis, reconciliations, and automated tools — as opposed to periodic [[Separate Evaluations]]. Together they implement the monitoring component of the [[COSO Internal Control Framework]] (see [[Monitoring of Internal Control]]).

Key facts:

- COSO's guidance recommends using ongoing monitoring in place of separate evaluations **where feasible**, because it detects problems closer to when they occur; separate evaluations periodically reconfirm that ongoing monitoring remains effective.
- Because it is built into operations, ongoing monitoring supports "building in" rather than "adding on" monitoring — a key efficiency response to [[SOX Section 404]] cost pressure.
- SEC Release 33-8810 accepts evidence from ongoing monitoring (including self-assessment and management's daily interaction with controls) for management's evaluation in lower-risk areas, with direct testing reserved for higher-risk areas.
- Automated forms of ongoing monitoring evolved into [[Continuous Controls Monitoring]] (threshold checks, duplicate-payment detection, [[Segregation of Duties]] analysis).
