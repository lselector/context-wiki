Oracle is a major enterprise software vendor whose ERP and GRC
cloud products are widely used to operationalize SOX controls. It
acquired [[NetSuite]] in 2016.

Key facts:
- Oracle Fusion Cloud Risk Management and Compliance (RMC) uses
  embedded AI and 300+ prebuilt algorithms for continuous SoD,
  access, configuration, and transaction monitoring supporting
  [[ICFR]] and [[SOX Section 404]]; AI agents (Certification
  Advisor, ERP Access Assistant, Assurance Advisor) generate
  audit-ready rationale for [[User Access Review]] decisions.
- Advanced Access Controls (AAC) provides
  [[Continuous Controls Monitoring]] of [[Access Controls]] and
  [[Segregation of Duties]] with a pre-built library drawing on
  6,000+ ERP Cloud entitlements, fine-grained full-path scans,
  simulation, and role optimization.
- Advanced Financial Controls monitors actual transactions —
  usable as [[Compensating Controls]] for accepted SoD conflicts.
